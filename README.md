# Project Image Hosting Repository

This repository is used to store and manage images for my projects. It serves as a centralized location for screenshots, diagrams, and other visual assets that are referenced across multiple repositories.

* Each project has its own folder inside the `/images` directory.
* Images are grouped by project to keep assets organized and easy to maintain.
